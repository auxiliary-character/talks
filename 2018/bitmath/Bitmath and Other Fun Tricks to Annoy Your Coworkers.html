Bitmath?

Code involving bitwise operations.

Computers use binary for math.

Digits are in the form of bit*(2^place) instead of digit*(10^place).

Binary digits (bits) hold boolean values.

Integers are arrays of bits.

Bitwise operations

  • Bitwise versions of boolean operations.
    • And: &
    • Or: |
    • Xor: ^
    • Not: ~
  • Bitshifting.
    • Left: <<
    • Right: >>

  • Xor

  • Not

Boolean Operations on integers

In C (and a few other languages), any non-zero integers are truthy.

Boolean not (! or not) is equavalent to checking equality to zero.

Returned booleans are either 0 (true) or 1 (false).

Bitwise operations are allowed on boolean values.

Fizzbuzz

Very simple problem, meant to test for baseline programming ability.

Divisible by 3 gets "Fizz"

Divisible by 5 gets "Buzz"

Divisible by both gets "Fizzbuzz"

Otherwise just print n.

#include 
#include 

int main(){
    for(unsigned int i=1; i<=100; ++i){
        bool div_3 = i % 3 == 0;
        bool div_5 = i % 5 == 0;
        if(div_3 && div_5){
            puts("Fizzbuzz");
        }
        else if(div_3){
            puts("Fizz");
        }
        else if(div_5){
            puts("Buzz");
        }
        else{
            printf("%u\n", i);
        }
    }
    return 0;
}

Rock Paper Scissors Lizard Spock

#!/usr/bin/env python3
from random import randint

options = ["rock","paper","scissors","lizard","spock"]
magic = 5843500

def game_iteration():
    user_in = input("Rock, paper, scissors, lizard, spock?\n").lower()

    if user_in == "exit":
        return False

    if not user_in in options:
        print("Bad input.")
        return True

    user = options.index(user_in)
    ai = randint(0,4)
    print("Player: ",options[user].capitalize())
    print("Computer: ",options[ai].capitalize())

    if user == ai:
        print("Tie")
    else:
        if (magic>>(user*5+ai))&1:
            print("You win!")
        else:
            print("You lose.")

    return True


def main():
    while game_iteration():
        pass

if __name__ == "__main__":
    main()
